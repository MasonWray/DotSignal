# xSignal

### Color Codes
| Color   | Command | RGB                  |
| ------- | ------- | -------------------- |
| Red     | `r`     | `rgb(255, 0, 0)`     |
| Green   | `g`     | `rgb(0, 255, 0)`     |
| Blue    | `b`     | `rgb(0, 0, 255)`     |
| Cyan    | `c`     | `rgb(0, 255, 255)`   |
| Magenta | `m`     | `rgb(255, 0, 255)`   |
| Yellow  | `y`     | `rgb(255, 255, 0)`   |
| White   | `w`     | `rgb(255, 255, 255)` |

| 50%       | `5`     | 125   |
| 60%       | `6`     | 150   |
| 70%       | `7`     | 175   |
